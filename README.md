# NLP-Time-Series-Neural-Networks-week9
## Customer Review Analytics Pipeline
A multi-technique data science project analyzing Amazon customer review data using NLP, time series forecasting, and neural networks to generate business insights from a single real-world dataset.

## Overview
Built as part of the Flatiron School AI & Data Science Bootcamp. Used a single real-world dataset to practice NLP, time series analysis, and deep learning within one project.

## What I Built
NLP Analysis: Preprocessed and analyzed review text to extract themes, word frequencies, and bigrams
Time Series Forecasting: Modeled customer activity and satisfaction trends over time using ARIMA
Neural Network: Built a text classification model using TF-IDF vectorization and Keras to predict customer ratings

## Tools & Libraries
Python, Pandas, NumPy, Matplotlib, Seaborn, NLTK, statsmodels, scikit-learn, TensorFlow, Keras

## Key Techniques
NLP: Tokenization, stopword removal, lemmatization, n-gram analysis
Time Series: ARIMA modeling, ADF stationarity testing, ACF/PACF analysis, seasonal decomposition
Neural Networks: TF-IDF vectorization, Keras dense layers, multi-class classification
